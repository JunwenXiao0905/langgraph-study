# LangGraph Study

This project uses `uv` and starts with a minimal `StateGraph` example.

## Run

```bash
uv run lg-tutorial
```

## First goal

The first example teaches:

- shared state with `TypedDict`
- nodes as plain Python functions
- edges from `START` to `END`
- `compile()` and `invoke()`
